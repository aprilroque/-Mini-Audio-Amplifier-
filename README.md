# Project1-Mini-Audio-Amplifier

# Description
A compact soldered mini amplifier using the LM386 IC capable of amplifying audio from mobile phones or laptops into a small speaker output.

# Features
- Portable design
- Low power consumption
- Direct soldered circuit
- Compact and durable

# Components Used
- LM386 IC
- Capacitors
- 8Ω Speaker
- 9V Battery

# Technologies Used
- Analog Electronics
- Audio Amplification
- Circuit Soldering
